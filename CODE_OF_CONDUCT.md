# 行動規範

## 目的

このプロジェクトはオープンで歓迎的なコミュニティを目指しています。

## 適用範囲

この行動規範は、Issue、プルリクエスト、コードレビューなど、プロジェクトに関するすべての活動に適用されます。

## 期待される行動

- 敬意を持った建設的なコミュニケーション
- 異なる視点や経験の尊重
- 批判を個人攻撃ではなく改善提案として受け入れる
- コミュニティの利益を優先する

## 許容されない行動

- 性的な言葉や画像の使用
- 荒らし、侮辱的なコメント、個人攻撃
- ハラスメント（公的・私的を問わず）
- 他者の個人情報の無断公開

## 違反の報告

行動規範に違反する行為を見かけた場合は、[GitHub Security Advisories](https://github.com/semba-yui/webpify/security/advisories/new) を通じて報告してください。

すべての報告は機密として扱われます。

## 措置

メンテナーは違反行為に対して以下の措置を取る権利があります：

- コメントの編集・削除
- 一時的または永久的な参加禁止

## 帰属

この行動規範は [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) に基づいています。
