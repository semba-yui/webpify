# Product Overview

webpify は画像ファイルを WebP 形式に変換するツールです。

## Core Capabilities

- **画像変換**: 各種画像フォーマット（PNG、JPEG、GIF など）を WebP 形式に変換
- **バッチ処理**: 複数の画像ファイルを一括で変換
- **品質制御**: 変換時の品質パラメータを調整可能

## Target Use Cases

- Web 開発者が画像を WebP 形式に最適化
- 画像アセットのバッチ変換による容量削減
- CI/CD パイプラインでの自動画像最適化

## Value Proposition

- シンプルな CLI インターフェースで手軽に画像変換
- WebP 形式により画像サイズを大幅に削減
- Web パフォーマンスの向上に貢献
